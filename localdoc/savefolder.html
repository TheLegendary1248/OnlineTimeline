usersaves Folder Structure

usersaves This is the main folder for all of your data after it has been processed by the program
<year> This folder contains all relevant data for the given year
<month> This folder contains all relevant data for the given month
<day>.json

This file should contain relevant information for the program about the day given

objects
global
<object>

This folder should contain relevant information for the given object.

The expected context is that each should represent a person's account or sort of entity outside of each platform and link to them

platforms
<platform>
<object>

This folder should contain relevant information for the given object.

The expected context is that each should represent a person's account or sort of entity on the given platform

path_reference

This folder contain json files that tell the program where your data is located and their format

[UNDECIDEDNAME].json
